# Servidor REST con Express + GraphQL + MongoDB Atlas + Mongoose

### Para instalar los paquetes necesarios usar:

``` npm install ```

### Para usar en local ejecutar:

``` npm run dev ``` or ``` npm start ```

### CRUD del modelo Product:

``` 
  title: {
    type: String,
    required: true,
  },
  description: {
    type: String,
    required: true,
  },
  price: {
    type: Number,
  }, 

```
